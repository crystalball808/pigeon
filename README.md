# Pigeon 🕊️
Pigeon is a terminal-based chat application.

## Features
- **Fullstack**: the project has both server and client modules
- **TUI**: use your terminal to chat with other users
- **Rust**: the application is memory-safe
## Stack
- **Tokio**: All the network stuff
- **Ratatui**: TUI library for client
## How to use
1. Launch a server module of the application
```shell
cargo run --bin server
ls
```
2. In separate terminal windows, launch multiple instances of a client module
```shell
cargo run --bin client
```
3. Enjoy chatting!
